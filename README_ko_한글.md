# YouTube 대본 (.txt) → SRT 변환기

이 Python 스크립트는 YouTube 스타일 대본 텍스트 파일을 `.srt` 자막 파일로 변환합니다. 타임스탬프를 표준화하고, 종료 시간을 생성하며, 순차적인 블록 번호를 삽입합니다.
(사용자는 먼저 YouTube 동영상에서 대본을 복사하여 .txt 파일로 저장해야 합니다.)

## 기능

- `m:ss` / `mm:ss`와 같은 짧은 타임스탬프를 완전한 `HH:MM:SS.fff` 형식으로 변환합니다 (예: `1:05` → `00:01:05.000`).
- 각 타임스탬프 줄에 대해 완전한 SRT 스타일 시간 범위(`시작 --> 종료`)를 생성합니다.
- 각 큐의 종료 시간을 다음 큐 시작 시간보다 **10밀리초 앞당겨** 설정합니다.
- 마지막 타임스탬프에 후속 타임스탬프가 없는 경우, 기본 지속 시간으로 **3초**(3000밀리초)를 할당합니다.
- 블록 사이에 빈 줄을 두고 SRT 블록 번호(`1`, `2`, `3`, ...)를 삽입합니다.

## 요구 사항

- Python 3.x (타사 패키지 없음; Python 표준 라이브러리만 사용).

## 입력 형식 (예상)

스크립트는 YouTube 동영상 설명란의 "자막 보기" 기능에서 볼 수 있고 복사할 수 있는 것과 동일한 타임스탬프 행과 자막 텍스트 행이 포함된 `.txt` 파일을 예상합니다. 예시:

```txt
0:01
안녕하세요

00:00:05.000
또 다른 줄
```

참고 사항:

- 타임스탬프 줄은 `m:ss` / `mm:ss` 형식 또는 완전한 `HH:MM:SS.fff` 형식일 수 있습니다.
- 타임스탬프가 아닌 줄은 자막 텍스트로 처리되어 변경 없이 그대로 전달됩니다.


## 출력

- 입력 파일 옆에 `.txt` 확장자를 `.srt`로 변경하여 `.srt` 파일을 생성합니다.
- 출력된 자막은 다음과 같습니다:

```txt
1
00:00:01.000 --> 00:00:04.990
안녕하세요
```


## 실행 방법

1. 스크립트를 저장합니다(예: `Ko_FormatSubtitleTimestamps.py`).
2. 터미널에서 실행합니다:
```bash
python Ko_FormatSubtitleTimestamps.py
```

3. 프롬프트가 표시되면 `.txt` 형식의 대본 파일 전체 경로를 붙여넣으세요.
    - 경로가 따옴표로 묶여 있어도 문제없습니다. 스크립트가 자동으로 주변 `"`를 제거합니다.

## 참고 사항 및 제한 사항

- 각 타임스탬프 행이 새로운 자막 큐의 시작을 표시한다고 가정합니다.
- 최종 큐 지속 시간은 3초로 고정됩니다. 다른 기본값을 원하면 `add_ms(current_ts, 3000)`을 변경하세요.
- 사용된 타임스탬프 형식은 `HH:MM:SS.fff`(점 + 밀리초)로, 많은 도구에서 수용하지만 일부 엄격한 SubRip 워크플로는 `HH:MM:SS,mmm`(쉼표)를 기대합니다.


## 사용자 정의

- 기본 마지막 큐 지속 시간: `add_ms(current_ts, 3000)` 내 `3000`을 다른 밀리초 값으로 변경하세요.
- 다음 큐 전 간격: 10ms보다 크거나 작은 간격을 원한다면 `subtract_ten_ms()` 로직을 수정하세요.
